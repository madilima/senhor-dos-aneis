A Java-based battle simulator inspired by The Lord of the Rings, designed to demonstrate Object-Oriented Programming principles such as inheritance, polymorphism, abstraction, and encapsulation.

The project simulates battles between the Fellowship of the Ring and the forces of Sauron on a linear map, where characters move, attack, and interact based on their race, class, and attributes.

⚔️ Key Features

Character System

Implementation of iconic characters like Aragorn, Gandalf, Legolas, Gimli, Boromir, Saruman, Orcs, Goblins, and Uruk-hai

Each character has attributes such as strength, agility, intelligence, and constitution

Race Abilities

Humans can age

Elves can speak Elvish

Dwarves can drink

Maiar can resurrect

Monsters can only grunt

Combat Classes

Warriors deal melee damage to nearby enemies

Archers attack enemies from distance

Mages deal area damage to all enemies

Battle Simulation

Turn-based system where characters attack and move

Movement direction depends on faction (Fellowship vs enemies)

Victory Conditions

The Fellowship wins if a member reaches the end of the map

Sauron wins if all Fellowship members are defeated

🧪 Testing

Includes unit tests with JUnit to validate battle scenarios, movement rules, and victory conditions.
